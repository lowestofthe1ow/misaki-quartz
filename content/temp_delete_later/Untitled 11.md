Program Counter
– Address of the next instruction
• Instruction Register
– Stores the instruction fetched from the memory
• Data Register
– Stores the data read from memory
• ALU
– Performs math operation (logic, arithmetic, comparison)
• Accumulator
– Contains the result from a math operation
• Data Address register
– Contains address in memory for data to be read or written
• Instruction Decoder
– Decodes instruction and produces necessary control to TCL
• Timing and Control Logic
– Synchronizes events and produces signal to devices (memory or I/O)
