Data science - derives insights and knowledge from both structured and unstructured data
Big Data - generation of a lot of data thanks to Internet connectivity

Volume, variety, velocity, veracity, value - V's of big data
Volume - V of big data that refers to the sheer amount of generated data
Variety - V of big data that refers to the diversity of data types and sources
Velocity - V of big data that refers to the speed at which data is generated, processed, and analyzed
Veracity - V of big data that refers to the authenticity and credibility of data (that is, its quality)
Value -  V of big data that refers to the added value for various stakeholders

Descriptive, Exploratory, Inferential, Predictive, Causal, Mechanistic - Types of data analysis
Descriptive - Type of data analysis, "How many COVID cases are there in the Philippines?"
Exploratory - Type of data analysis, "Is the trend of COVID cases going upwards or downwards?"  
Inferential - Type of data analysis, "Is the daily change of cases in NCR the same as the change of cases in Region 4?"
Predictive - Type of data analysis, "How many COVID cases will happen in the next month?"
Causal - Type of data analysis, "If mask wearing was not implemented in the Philippines, how will it affect the number of cases in the country?"
Mechanistic - Type of data analysis, "How does mask wearing reduce the transmission of COVID?"

Data → Information → Insight → Imperative → repeat  - Data analytics pipeline
Data - Data analytics pipeline, creation of data from operations or transactions
Information - Data analytics pipeline, consolidation of relevant data to answer a question or a problem (warehousing, business intelligence, descriptive)
Insight - Data analytics pipeline, finding patterns to answer why it happened and what could happen next (diagnostics, predictive)
Imperative - Data analytics pipeline, development of various solutions to suggest what should be done next (prescriptive)

Structured, semi-structured, unstructured - Forms of data
Structured data - Form of data with identifiers (easy searching, e.g. relational databases)
Semi-structured data - Form of data with identifiers, not relational but may be loaded into a relational database (e.g. XML, CSV, JSON)
Unstructured data - Form of data with no labels (raw format, e.g. video, sound, text, PDF)

Qualitative, quantitative - Types of variables
Qualitative - Type of variable that indicates a characteristic or quality
Quantitative - Type of variable that indicates an amount or quantity of a variable
Continuous or discrete - Quantitative variables can be:
Categorical - Qualitative variables can be:

Nominal scale - Scale for categorical variables with no intrinsic ordering
Ordinal scale - Scale for categorical variables with ordering
Interval scale - Quantitative scale, raw magnitude, equal intervals between adjacent units, difference between values is meaningful, no absolute zero point
Ratio scale - Quantitative scale, raw magnitude, equal intervals between adjacent units, difference and ratio between values is meaningful, absolute zero point exists

Response, selection, confirmation, social desirability, publication, data dredging - Types of data bias
Response bias - Data bias where content generated by humans is subject to their own opinions
Selection bias - Data bias where data is subjectively selected (only people interested answer surveys)
Confirmation bias - Bias towards proving a predetermined assumption
Social desirability bias - Bias when survey participants respond in a manner that they think will be favorably viewed
Publication bias - Less significant results are omitted in publication
Data dredging - Presenting statistically significant results without a pre-existing hypothesis (failing to acknowledge that correlations can occur by chance)

Probability distribution - Specifies what values can be taken by data points and how frequently these values will occur

Mean, median, mode - Measures of central tendency; gives an idea of what the common observation is like
Mean - Measure of central tendency, simply the average; interval or ratio data only
Median - Measure of central tendency, middle value when data is ordered; any quantitative data
When data is skewed or with outliers - When do you use median instead of mean for quantitative data?
Mode - Measure of central tendency, most frequent value, best for categorical data

Mean < median < mode - Relationship of mean, median, mode when left/negative skew
Mean = median = mode - Relationship of mean, median, mode when normal distribution
Mode < median < mean - Relationship of mean, median, mode when right/positive skew

Range, variance - Measures of dispersion; how spread out the data is
Range - Measure of dispersion, difference between largest and smallest value
Variance - Measure of dispersion, average squared differences from the mean
Standard deviation - Square root of variance, "how different is it from the mean?"

Domain, abstraction, idiom, algorithm - Four levels of visualization design

Abstraction - Level of visualization design: translating domain knowledge to visualization

Expressiveness - Matching the channel to the data type
Effectiveness - Most important attributes with highest ranked channel

Weber's law - Ratio of increment to background should be constant (as the human perceptual system operates with relative judgments rather than absolutes)

Overview first, details on demand - Visualization design rule of thumb for interactivity: Overview or details first?

Color deficiency - Reduce color dimensionality
Viridis color scale - Color scale that is colorful, perceptually uniform, colorblind-friendly, and has monotonically increasing luminance



Key - Independent attribute; unique index to look up items
Value - Dependent attribute; value of cell

Scatterplot - Type of plot with no keys, values only; data is 2-quant attributes
Find trends, outliers, distribution, correlation, clusters - Task of a scatterplot

Line chart - Type of plot with one key and one value used to find a trend

Bar chart - Type of plot with one key and one value used for comparison or looking up values
Categorical key, quantitative value - In a bar chart, the types of the key and value are:

Bar chart if key is categorical, line chart if ordered - When to use a bar chart vs line chart

Stacked bar chart - Type of plot with two keys and one value; two categorical keys; expresses a part-to-whole relationship

Heatmap - Type of plot with two keys, one value (two categorical, one quantitative); finds clusters, outliers

Pie chart - Type of plot with one categorical key, one quantitative value; parts of a whole; area marks with ANGLE channel

Polar area chart - Type of plot with one categorical key, one quantitative value; parts of a whole; area marks with LENGTH channel

Normalized stacked bar chart - Stacked bar chart but normalized to full vertical height; similar to "densely-packed pie chart"

Choropleth map - Spatial geometry; one quantitative attribute per region
