## Identity

$A \cap \mathcal{U} = A$

$A \cup \emptyset = A$

<!--ID: 1707668756755-->

## Domination

$A \cap \emptyset = \emptyset$

$A \cup \mathcal{U} = \mathcal{U}$

<!--ID: 1707668756760-->

## Idempotence

$A \cap A = A$

$A \cup A = A$

<!--ID: 1707668756764-->

## Complement

$A \cap \bar{A} = \emptyset$

$A \cup \bar{A} = \mathcal{U}$

<!--ID: 1707668756768-->

## Complementation

$\bar{\bar{A}} = A$

<!--ID: 1707668756771-->

## De Morgan's

$\bar{A \cap B} = \bar{A} \cup \bar{B}$

$\bar{A \cup B} = \bar{A} \cap \bar{B}$

<!--ID: 1707668756775-->

## Commutative

$A \cap B = B \cap A$

$A \cup B = B \cup A$

<!--ID: 1707668756779-->

## Associative

$A \cap (B \cap C) = (A \cap B) \cap C$

$A \cup (B \cup C) = (A \cup B) \cup C$

<!--ID: 1707668756782-->

## Distributive

$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$

$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$

<!--ID: 1707668756785-->

## Absorption

$A \cap (A \cup B) = A$

$A \cup (A \cap B) = A$
