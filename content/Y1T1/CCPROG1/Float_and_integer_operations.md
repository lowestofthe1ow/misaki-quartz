## Operator: + (addition)

|**Rule**|**Example**|
|---|---|
|int + int = int|5 + 2 = 7|
|int + float = float|5 + 2.0 = 7.0|
|float+ int = float|5.0 + 2 = 7.0|
|float+ float = float|5.0 + 2.0 = 7.0|
