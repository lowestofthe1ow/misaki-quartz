## CSMATH1 LQ2 Review

**I.** Select the letter corresponding to the best answer.

1. A secant line for a function has a slope $\frac{\Delta y}{\Delta x}$. Which of the following best describes a **tangent** line for that function?

**A.** $\lim_{x \to 0} \frac{\Delta y}{\Delta x}$
**B.** $\lim_{y \to 0} \frac{\Delta y}{\Delta x}$
**C.** $\lim_{\Delta y \to 0} \frac{\Delta y}{\Delta x}$
**D.** $\lim_{\Delta x \to 0} \frac{\Delta y}{\Delta x}$

2. Which of the following describes the derivative of a function $f(x)$?

**A.** The slope of a secant line intersecting the graph at $x$
**B.** The $y$-intercept of a secant line intersecting the graph at $x$
**C.** The $y$-intercept of a line tangent to the graph at $x$.
**D.** The slope of a line tangent to the graph at $x$.

3. **True or false.** If a function is continuous at a point $x$, then it is differentiable at that point.

**A.** True
**B.** False; there is no relationship between differentiability and continuity.
**C.** False, but if a function is differentiable at a point, then it is continuous at that point.

4. A function $f(x)$ has the following property at a point $x_1$. $$\lim_{\Delta x \to 0} \frac{f(x_1 + \Delta x) - f(x_1)}{\Delta x} = +\infty$$What can be said about the line tangent to the graph of $f(x)$ at $x_1$?

**A.** The tangent line is a horizontal line.
**B.** The tangent line is a vertical line.
**C.** The tangent line does not exist.

5. **True or false.** The function given in the previous question is differentiable at $x = x_1$.

**A.** True
**B.** False

6. Let $f(x) = x^2 + \sin(x)$. What is the line tangent to the graph of $f(x)$ at $x=0$?

**A.** $y = -x$
**B.** $y = 3x$
**C.** $y = x$
**D.** $y = -3x$

7. At what point on the graph of $f(x) = x^2$ is the tangent line perpendicular to the line $2x - 4y + 3 = 0$?

**A.** $(-1, 1)$
**B.** $(1, 1)$
**C.** $(1, -1)$
**D**. $(-1, -1)$
